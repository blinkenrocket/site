# Blinkenrocket{.de,.com}

A simple website to present the blinkenrocket project and link to
external resources.

# Usage

On Debian or Ubuntu:

* `sudo apt install python python-jinja python3-lesscpy`

On other systems:

* Install python2.7
* `pip install lesscpy jinja2`

And then, no matter on which system:

`python build.py`
